The Crystal Collector Game

Click on a Crystal to determine the random value between 1 and 12. Each Crystal is given a random value at the start of each new game. The goal is to click each the crystals so that the added value is equal to random goal generated at the begining of each game. If the total value clicked is equal to the random goal, then you win. If it exceed the goal, you lost the game and begin again.
